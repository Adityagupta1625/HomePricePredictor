## HomePricePredictor
it is simple web app for housing price prediction based on certain parameters with the help of model trained through a sample dataset from kaggle.

### Tech Stck:
- flask
- Numpy
- Pandas
- MatplotLib
- Sklearn
- HTML
- CSS
- JavaScript
